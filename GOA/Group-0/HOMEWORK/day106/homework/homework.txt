რა არის vite?

Vite (ფრანგული სიტყვა "სწრაფი", გამოთქმული /vit/,
ისევე როგორც "veet") არის build ინსტრუმენტი, რომელიც 
მიზნად ისახავს უზრუნველყოს უფრო სწრაფი და მყუდრო განვითარების 
გამოცდილება თანამედროვე ვებ პროექტებისთვის. იგი შედგება ორი ძირითადი ნაწილისაგან:

დეველოპერის სერვერი, რომელიც უზრუნველყოფს მდიდარ ფუნქციების 
გაუმჯობესებას მშობლიურ ES მოდულებთან შედარებით, მაგალითად, 
ძალიან სწრაფი ცხელი მოდულის ჩანაცვლება (HMR).

build-ის ბრძანება, რომელიც აერთიანებს თქვენს კოდს Rollup-თან,
რომელიც წინასწარ არის კონფიგურირებული წარმოებისთვის მაღალოპტიმიზებული სტატიკური აქტივების გამოსატანად.

Vite არის მოსაზრება და გააჩნია გონივრული ნაგულისხმევი პარამეტრები.
წაიკითხეთ იმის შესახებ, თუ რა არის შესაძლებელი მახასიათებლების სახელმძღვანელოში.
ჩარჩოების მხარდაჭერა ან სხვა ინსტრუმენტებთან ინტეგრაცია შესაძლებელია Plugin-ის საშუალებით. 
კონფიგურაციის განყოფილება განმარტავს, თუ როგორ უნდა მოერგოს Vite თქვენს პროექტს საჭიროების შემთხვევაში.

რა არის React?

React საშუალებას გაძლევთ შექმნათ მომხმარებლის ინტერფეისი ცალკეული 
ნაწილებისგან, რომელსაც ეწოდება კომპონენტები. შექმენით თქვენი საკუთარი React კომპონენტები, როგორიცაა მინიატურა, 
LikeButton და ვიდეო. შემდეგ დააკავშირეთ ისინი მთელ ეკრანებად, გვერდებად და აპებად.
